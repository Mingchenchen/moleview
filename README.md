# moleview
Molecule + Viewer = MoleView <br/>
View your molecule anywhere and anytime!

## Installation

```sh
pip install moleview
```

**Requirements**
- Python 3
- NumPy
- Matplotlib

## Usage

```sh
moleview YOUR_COMPUTATIONAL_CHEMISTRY_FILE
```
